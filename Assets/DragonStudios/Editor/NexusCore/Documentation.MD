# Draconis Nexus v1.0.2

Draconis Nexus is the official toolkit for Dragon Studios Entertainment, providing essential tools for Unity development.

## Getting Started
1. Open the Draconis Nexus window from the Unity menu: `Window > Draconis Nexus`
2. Navigate through the different tabs to access various tools
3. Use the main hub for quick access to commonly used features

## Main Tabs
- **Nexus Center**: Central hub with quick access to all tools
- **Nexus Catalog**: Asset management tools (Prefab Search, Asset Importer)
- **Nexus Creator**: Content creation tools (Prefab Creator, Folder Creator, Dream Shaper)
- **Documentation**: Access to this documentation
- **Changelog**: Version history and updates

## Core Tools
1. **Documentation**: This help documentation
2. **Notepad**: Simple text editor for notes and reminders
3. **Folder Creator**: Create and manage folder structures
4. **To-Do List**: Task management with due dates
5. **Prefab Search**: Find and manage prefabs in your project
6. **Asset Importer**: Import and organize assets with previews
7. **Dream Shaper**: Place and manage prefabs in your scene

## Dream Shaper
    1. Click on the "Draconis Nexus" menu item
    2. Click on the "Dream Shaper" menu item
    3. Click on the "Add Prefab Slot" button
    4. Click on the "Active Prefab" dropdown menu Choose a prefab in the list
    5. Make sure you have clicked the Spawn "Button in the bottom left corner"
    6. Click any where on the scene to spawn the prefab.
    7. Click on the "Align to Normal" checkbox
    8. Set your spawn height offset
    9. Click on the "Rotate, Scale, or Move" button to rotate, scale, or move the prefab.

## Folder Creator
    1. Click on the "Draconis Nexus" menu item
    2. Click on the "Folder Creator" menu item
    3. Add a new folder by clicking on the "Add Folder" button
    4. Add a child folder by clicking on the "Add Child Folder" button
    5. Name the folders in the hierarchy
    6. To save the folder structure, click on the "Save" button
    7. To load a json folder structure, click on the "Load Preset" button

## To-Do List
1. Open the To-Do List from Nexus Center or the main menu
2. Click "Add Task" to create a new task
3. Fill in task details:
   - Title (required)
   - Description (optional)
   - Due date (optional)
4. Tasks are color-coded by due date:
   - Red: Overdue
   - Yellow: Due soon (within 3 days)
   - Green: Not due yet
5. Mark tasks as complete using the checkbox
6. Toggle between active and completed tasks using the "View Completed" button

## Asset Importer
1. Access from the Nexus Catalog tab
2. Click "Select Folder" to choose a source directory
3. View asset previews in the grid
4. Use the search bar to filter assets
5. Select assets to import
6. Click "Import Selected" to import assets to your project
